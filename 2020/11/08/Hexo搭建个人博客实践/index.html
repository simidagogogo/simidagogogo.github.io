Hexo搭建个人博客实践

Posted on 2020-11-08 In Tools
2020/11/07。记录通过GitHub+Hexo 方式，本人搭建的第一个独立个人博客网站。

简介

Hexo是一个简单、快速、强大的基于 Github Pages 的博客发布工具，支持Markdown格式，有众多优秀插件和主题，优点如下：

  1. 全是静态文件，访问速度快
  2. 免费搭建自由的个人博客，无需服务器
  3. 随意绑定个人域名
  4. 基于github的版本管理，数据安全，版本可回滚
  5. 博客内容可以轻松打包、转移、发布到其它平台

环境准备

工具准备

  • git：用于将静态页面部署到github上
  • Node.js 和 npm：需要通过npm安装Hexo
  • github账号：新建username.github.io仓库，并托管博客静态页面，同时也是网站地址

备注：node和npm安装方式有两种，官网下载node-vxx.xx.0.pkg，或者通过homebrew安装（需要先安装homebrew）

采坑排坑

1. 安装Homebrew报错

问题描述：报错Failed to connect to raw.githubusercontent.com port 443: Connection refused（GitHub的raw.githubusercontent.com域名解析被污染）

解决办法：修改hosts

2. 安装npm报错

问题描述：安装npm报错rollbackFailedOptional verb npm-session

解决办法：修改npm的资源镜像链接

3. https://xxx.github.io无法访问

问题描述：浏览器输入https://xxx.github.io，显示找不到 .github.io 的服务器IP地址。

解决方案1：在确定没有设置防火墙的情况下。 查询 自己的博客ip（查询栏输入username.github.io），根据搜索出来的ip地址， 在/etc/hosts 文件中增加如下代码：
