特征交叉代码实现

发表于 2020-12-12 分类于 Machine Learning
2020/12/12，记录特征交叉的代码实现。
背景

特征交叉的实现代码，即特征列表中的各个值进行笛卡尔积，写的比较精妙。

代码逻辑

对若干个特征（一般为2个），进行特征交叉。过程即对若干特征之间做笛卡尔积。时间复杂度为O(n^num)，其中num为交叉特征的个数。

详细过程

1、对indexes进行初始化

2、通过while循环，模拟特征交叉的笛卡尔积。

  • 跳过f_index为0的元素
  • 将各个特征的f_index和index，按顺序拼接后作为key进行hash，得到pos并添加进buf

3、模仿带进位加法（从低到高），进行排列组合

详细过程和代码实现

1、对indexes进行初始化

2、通过while循环，模拟特征交叉的笛卡尔积。

  • 跳过f_index为0的元素

  • 将各个特征的f_index和index，按顺序拼接后作为key进行hash，得到pos并添加进buf

3、模仿带进位加法（从低到高），进行排列组合

例子

假设rnfs.size() = 2。即两个特征之间做交叉。

f1 的 feature_list = [(1, 0.1), (2, 0.2), (3, 0.3), ..., (64, 0.9)]

f2 的 feature_list = [(1, 0.9), (2, 0.8), (3, 0.7), ..., (64, 0.1)]

indexes中的元素值，对应各个特征feature_list中元素位置。例如indexes(0) = 9，表示第一个特征中feature_list下标第9位元素。indexes的长度等于交叉特征的个数，即indexes.size() = rnfs.size() 。
